Time tracker

Небольшое приложение для отслеживания своего денежного ожидания от работы над проектом. Пользователь создает проект, указывает свою почасовую ставку и запускает трекер во время работы. Программа отслеживает время работы и умножает его на почасовую ставку, выводя итоговую сумму. Для написания этого приложения использовалась связка React + Redux

Task Manager

Веб-приложение для организации задач. Пользователи могут быть двух типов: администратор (может добавлять и удалять пользователей) и обычный пользователь. Приложение позволяет создавать задачи с разными параметрами (исполнитель, ответственное лицо, клиент, срок выполнения и приоритет) и менять их статус по обстоятельствам. При создании задачи происходит уведомление всех участников по электронной почте. Каждую задачу можно комментировать. Имеется система авторизации. Использованные технологии: node, express, mongodb, mongoose, react, react-router, passport, jsonwebtoken и nodemailer. Приложение развернуто на сервисе Heroku.

  • - Система аунтентификации пользователей(JWT)
  • - Создание, редактирование, перемещение, удаления, фильтрация задач
  • - Задаче можно присвоить исполнителя, ответственного, клиента
  • - Каждому пользователю связанному с задачей приходят уведомления о всех действиях производимых над задачей.
  • - Пользователи могут комментировать задачи
